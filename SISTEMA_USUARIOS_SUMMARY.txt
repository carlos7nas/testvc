📋 INSTRUÇÕES DE SETUP
═══════════════════════════════════════════════════════════════════════════════

1. Criar banco de dados SQL Server:
   - Execute o script fornecido em seu SQL Server Management Studio
   - Crie a database "mobgames_store"
   - Crie a tabela "Usuario" com os campos especificados

2. Configurar conexão (se necessário):
   - Edite DatabaseHelper.cs
   - Altere a string ConnectionString para seu servidor

3. Testar conexão:
   - Executar a aplicação
   - Abrir UserForm
   - Se conexão falhar, verificar:
	 * SQL Server está rodando?
	 * Banco de dados existe?
	 * Credenciais de autenticação estão corretas?

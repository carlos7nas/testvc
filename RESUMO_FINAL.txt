🚀 COMO USAR
═══════════════════════════════════════════════════════════════════════════════

PASSO 1 - CRIAR BANCO DE DADOS:
  Execute o script SQL fornecido em SQL Server Management Studio

PASSO 2 - CONFIGURAR CONEXÃO:
  Edite App.config se seu servidor for diferente de .\SQLEXPRESS

PASSO 3 - COMPILAR:
  Build → Build Solution (ou Ctrl+Shift+B)

PASSO 4 - TESTAR:
  Pressione F5 para executar a aplicação

PASSO 5 - USAR:
  UsuarioFormHelper.AbrirNovoUsuario()  // Novo usuário
  UsuarioFormHelper.AbrirEditarUsuario(id)  // Editar
  UsuarioFormHelper.CarregarUsuariosEmGrid(grid)  // Listar
  UsuarioFormHelper.DeletarUsuario(id, nome)  // Deletar

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Erro: "Não consegue conectar ao BD"
→ Verifique SQL Server, banco de dados, string de conexão

Erro: "Email já está registrado"
→ Use outro email ou delete o anterior

Erro: "Validação falhou"
→ Preencha Nome e Email corretamente

Erro: "Não consegue carregar foto"
→ Use formatos suportados (JPG, PNG, GIF, BMP)

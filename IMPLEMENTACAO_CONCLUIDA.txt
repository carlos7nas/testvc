💻 EXEMPLOS DE USO
═══════════════════════════════════════════════════════════════════════════════

NOVO USUÁRIO VIA FORMULÁRIO:
────────────────────────────────────────────────────────────────────────────────
if (UsuarioFormHelper.AbrirNovoUsuario())
{
	MessageBox.Show("Usuário criado com sucesso!");
}

EDITAR USUÁRIO VIA FORMULÁRIO:
────────────────────────────────────────────────────────────────────────────────
if (UsuarioFormHelper.AbrirEditarUsuario(usuarioId))
{
	MessageBox.Show("Usuário atualizado!");
}

LISTAR USUÁRIOS EM DATAGRIDVIEW:
────────────────────────────────────────────────────────────────────────────────
UsuarioFormHelper.CarregarUsuariosEmGrid(dataGridView1);

DELETAR USUÁRIO:
────────────────────────────────────────────────────────────────────────────────
if (UsuarioFormHelper.DeletarUsuario(usuarioId, nomeUsuario))
{
	MessageBox.Show("Deletado com sucesso!");
}

CRIAR USUÁRIO PROGRAMATICAMENTE:
────────────────────────────────────────────────────────────────────────────────
int novoId = UsuarioManager.SalvarUsuario(
	"João Silva", "joao@email.com", "(11) 98765-4321",
	"Rua A, 123", "123.456.789-00", "São Paulo", "01000-000"
);

OBTER USUÁRIO:
────────────────────────────────────────────────────────────────────────────────
var usuario = UsuarioManager.ObtenerUsuario(1);
if (usuario != null)
	MessageBox.Show($"Nome: {usuario.Nome}");

TESTAR CONEXÃO:
────────────────────────────────────────────────────────────────────────────────
if (DatabaseHelper.TestConnection())
	MessageBox.Show("Conexão OK!");

📋 BANCO DE DADOS - SCRIPT DE CRIAÇÃO
═══════════════════════════════════════════════════════════════════════════════

USE master 
GO 
IF EXISTS (SELECT * FROM sysdatabases WHERE name = 'mobgames_store') 
	DROP DATABASE mobgames_store 
GO 
CREATE DATABASE mobgames_store 
GO 
USE mobgames_store 
GO 

CREATE TABLE Usuario ( 
	id_usuario INT IDENTITY(1,1) PRIMARY KEY, 
	nome NVARCHAR(100) NOT NULL, 
	email NVARCHAR(150) NOT NULL UNIQUE, 
	senha_hash NVARCHAR(255) NOT NULL, 
	data_criacao DATETIME DEFAULT GETDATE(), 
	endereco NVARCHAR(150), 
	bairro NVARCHAR(100), 
	cidade NVARCHAR(100), 
	cep NVARCHAR(10),
	telefone NVARCHAR(15),
	cpf NVARCHAR(20)
)
GO

📞 SUPORTE TÉCNICO
═══════════════════════════════════════════════════════════════════════════════

Para resolver problemas comuns:

1. "Não consegue conectar ao BD"
   → Verifique se SQL Server está rodando
   → Confirme que o banco 'mobgames_store' existe
   → Teste com SQL Server Management Studio

2. "Email já está registrado"
   → Email deve ser único no banco
   → Tente com outro email

3. "Erro ao carregar foto"
   → Formato deve ser: JPG, PNG, GIF, BMP
   → Tamanho máximo: Sem limite definido
   → Tente colocar a imagem em outro formato

4. "Erro SQL: Table not found"
   → Execute o script SQL fornecido
   → Certifique-se que a tabela 'Usuario' foi criada
